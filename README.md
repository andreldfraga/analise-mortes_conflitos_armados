# ⚔️ Análise de Mortes em Conflitos Armados

Este projeto tem como objetivo analisar o número acumulado de mortes em conflitos armados ao longo do tempo, utilizando dados disponibilizados pelo **[Our World in Data](https://ourworldindata.org/grapher/cumulative-deaths-in-armed-conflicts)**.  
A análise busca identificar padrões temporais, regionais e comparativos, além de gerar visualizações informativas sobre a intensidade e a distribuição dos conflitos.

---

## 🎯 Objetivos

- Explorar a evolução temporal das mortes em conflitos armados.  
- Comparar a intensidade dos conflitos entre diferentes regiões e períodos.  
- Produzir visualizações e estatísticas descritivas sobre os dados.  
- Criar uma base analítica replicável para futuras atualizações ou estudos.

---

## 🧰 Tecnologias Utilizadas

- **Python**: pandas, numpy, matplotlib, seaborn, geopandas  
- **Jupyter Notebook**: para exploração e visualização dos dados  
- **Git / GitHub**: controle de versão e colaboração
